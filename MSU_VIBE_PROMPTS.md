### 등장 요소
- **플레이어 — 직업 3종 선택**: 전사 / 마법사 / 궁수. 직업마다 스탯·외형·스킬이 다름.

- **유물 5개**: 던전 곳곳에 배치. 모두 모으면 수호자 보스(머쉬맘) 등장.
- **포탈**: 보스 처치 시 활성화되는 탈출구.

### 직업(클래스)
| 직업 | HP/MP/ATK | 특징 | 스킬 |
|---|---|---|---|

### 배틀 명령 (4메뉴 + 스킬 서브메뉴)
1. **공격** — 기본 데미지 (공격력 + 0~3), 치명타 시 ×1.6
2. **스킬** — 직업별 스킬 목록 서브메뉴 (MP 소모, 다단히트/독/회복/버프)
3. **물약** — HP 35 회복(턴 소모), 3개 제한
4. **도망** — 50% 확률 탈출(보스는 도망 불가)

### 상태이상
- **독(poison)**: 4턴간 매 턴 시작 시 6 데미지. 마법사 포이즌미스트, 독 계열 몬스터·보스가 부여.

### 성장 공식
- 경험치 누적 → 레벨업 시 최대HP +11, 공격력 +2, 최대MP +3, HP/MP 완전 회복
- 다음 레벨 필요 경험치 = 직전 × 1.5 (시작 22)
- 시작 물약 4개 / 물약 회복 +42

## 2. 단계별 바이브 코딩 프롬프트

> 💡 팁: Agent 8은 "한 번에 하나의 기능"을 요청할수록 정확하다. 아래를 **순서대로** 입력하고,
> 각 단계가 동작하는 것을 확인한 뒤 다음 단계로 넘어간다. 메이플 에셋은 MSU Space 라이브러리에서
> 검색해 지정한다.

### STEP 0 — 직업 선택 화면
```
게임 시작 시 직업 선택 화면을 보여줘. 전사 / 마법사 / 궁수 세 직업을 카드로 보여주고,
각 카드에 직업 아이콘(메이플 캐릭터), 스탯(HP/MP/공격력), 스킬 목록을 표시해.
플레이어가 직업을 고르면 그 스탯·외형·스킬로 게임이 시작된다.
- 전사: HP65 MP14 ATK11, 근접 강타형
- 마법사: HP45 MP26 ATK10, 마법·자가회복형
- 궁수: HP54 MP18 ATK10, 다단히트·치명타형
```

### STEP 1 — 프로젝트와 필드(탑다운) 만들기
```
메이플스토리 IP를 사용한 탑다운(위에서 내려다보는 시점) 2D 게임을 만들어줘.
- 25x17 타일의 던전 맵 한 화면. 벽 타일은 통과 불가, 바닥 타일은 이동 가능.
- 선택한 직업의 메이플 캐릭터를 플레이어로 배치하고, WASD와 화살표 키로 4방향 이동.
- 벽과 충돌 처리. 카메라는 화면 고정.
- 헤네시스 던전 느낌의 배경 타일과 BGM을 메이플 에셋에서 사용.
```

### STEP 2 — 유물과 포탈(목표) 배치
```
맵 곳곳에 '고대 유물' 아이템 5개를 배치해줘. 플레이어가 닿으면 획득되고, 화면 상단에
"유물 0/5" 카운터가 올라가며 반짝이는 획득 이펙트가 나오게 해줘.
유물 5개를 모두 모으면 맵 한쪽의 포탈이 활성화(잠금 해제)되고, 활성화된 포탈에
플레이어가 들어가면 "탈출 성공" 승리 화면을 띄워줘.
```

### STEP 3 — 필드 몬스터와 "충돌 시 배틀 진입"
```
맵에 메이플 몬스터(달팽이, 주황버섯, 슬라임)를 여러 마리 배치해줘. 몬스터는 천천히 배회하다가
플레이어가 가까워지면 다가온다. 플레이어가 몬스터와 부딪히면, 포켓몬처럼 화면이 전환 연출
(가로 셔터/페이드)과 함께 '배틀 화면'으로 바뀌게 해줘. 배틀이 끝나고 필드로 돌아오면 잠깐 동안
같은 몬스터와 다시 충돌하지 않도록 무적 시간을 줘.
```

### STEP 4 — 턴제 배틀 화면 (포켓몬식)
```
배틀 화면을 만들어줘.
- 오른쪽 위에 적 몬스터 스프라이트와 이름/HP바, 왼쪽 아래에 플레이어 캐릭터와 이름/Lv/HP바.
- 하단에 메시지창. "야생의 OO이(가) 나타났다!" 같은 대사를 한 줄씩 보여주고 Enter/Space로 넘긴다.
- 메시지가 끝나면 명령 메뉴 4개: 공격 / 스킬 / 물약 / 도망. ↑↓로 선택, Enter로 확정.
- '스킬'을 고르면 직업별 스킬 목록(이름·MP·설명)이 서브메뉴로 뜨고, Esc로 뒤로 갈 수 있다.
- 좌측에 HP/MP/물약/공격력버프 상태를 표시.
- 메이플 몬스터/캐릭터 스프라이트와 배틀 BGM을 에셋에서 사용.
```

### STEP 5 — 배틀 규칙(데미지·턴 순서·상태이상)
```
배틀 로직을 만들어줘. 턴제: (독 처리) → 플레이어 행동 → (독 처리) → 적 반격 순서.
- 공격: 공격력 + 0~3 데미지. 치명타(직업별 확률) 시 ×1.6.
- 스킬: 직업별 스킬. MP 소모. 종류 = 단일공격(배수)/다단히트(2~3회)/독 부여/자가회복/공격력 버프.
  MP 부족하면 사용 불가 메시지.
- 물약: HP 42 회복(최대치 초과 불가), 보유 4개 제한, 사용하면 한 턴 소모.
- 도망: 50% 확률로 탈출. 실패하면 적이 공격. 단, 보스에게는 도망 불가.
- 독(상태이상): 4턴간 매 턴 시작 시 피해(플레이어 4, 적 6). 플레이어/적 모두 걸릴 수 있다.
- 적 공격: 적 공격력 + 0~2 데미지. 독 계열 몬스터는 독 포자, 강력 계열은 가끔 묵직한 일격(×1.6).
- 적 HP 0 → 승리. 플레이어 HP 0 → 게임 오버.
모든 행동을 메시지창에 출력하고, 피격 시 화면 흔들림/번쩍임 연출.
```

### STEP 6 — 몬스터 12종 + 성장(레벨업)
```
메이플 몬스터를 티어별로 배치해줘 (아래 밸런스 표 참고). 약한 몬스터부터 강한 몬스터까지 12종.
- 일부는 독 계열(초록버섯·좀비버섯)로 독 포자 공격, 일부는 강력 계열(다크스텀프·와일드보어)로 강타.
배틀 승리 시 경험치 획득, 누적이 필요치를 넘으면 레벨업.
레벨업 시 최대HP +11, 공격력 +2, 최대MP +3, HP/MP 완전 회복. 필요 경험치는 매 레벨 1.5배(시작 22).
승리한 몬스터는 필드에서 사라진다.
```

### STEP 7 — 보스(머쉬맘)와 클리어 조건
```
유물 5개를 모두 모으면 탈출 포탈 근처에 보스 '머쉬맘'이 등장한다(경고 메시지 + 등장 이펙트).
- 머쉬맘: HP 200, 공격 17, 경험치 140. 도망 불가.
- 보스는 3턴 주기로 패턴 변경: ①기본 공격 ②포자 폭발(×1.6, 독 부여) ③강타(×1.3).
- 머쉬맘을 쓰러뜨리면 포탈이 열리고, 포탈에 들어가면 다음 층(B2)으로.
- 보스를 잡기 전 포탈에 닿으면 "머쉬맘을 먼저 처치하라" 안내.
```

### STEP 8 — 다층 던전(B2)과 최종 보스 자쿰
```
던전을 여러 층으로 만들어줘.
- B1에서 머쉬맘을 처치하고 포탈에 들어가면, '층 클리어' 화면이 뜨고(HP·MP 완전 회복 + 물약 2개 보상)
  더 깊은 B2 '망각의 지하'로 내려간다. 캐릭터 레벨·스탯은 유지된다.
- B2는 다른 맵 구조 + 더 강한 몬스터 풀: 커즈아이·와일드카고·다크예티·주니어발록 등(아래 표 참고).
- B2의 유물 5개를 모으면 최종 보스 '자쿰'이 등장. 자쿰은 4턴 주기 패턴:
  ①대지의 분노(×1.4) ②흑염 브레스(×1.3+독) ③팔 휘두르기(×1.1) ④융합 폭발(×1.8).
- 자쿰을 쓰러뜨리고 포탈에 들어가면 최종 "탈출 성공" 승리 화면.
(추가 확장 시 B3·B4와 더 강한 보스를 같은 패턴으로 이어붙이면 됨.)
```

```
- 시작 화면 → 직업 선택 화면 → 게임 순서.
- 종료 화면: 승리("머쉬맘을 물리치고 탈출! 최종 직업 Lv.N") / 패배("게임 오버") + "다시 하기".
- 메이플 특유의 밝고 귀여운 톤, 효과음(공격/스킬/획득/레벨업/보스등장)을 에셋에서 입혀줘.
```

## 4. 제출 시 어필 포인트 (심사 대비)

- **메이플 IP 활용도**: 달팽이 3종·버섯류·슬라임·스텀프·와일드보어·보스 머쉬맘 등 상징적 몬스터 13종 + 헤네시스 던전 무드 + 전사/마법사/궁수 직업
- **완결된 게임 루프**: 직업 선택 → 탐험 → 배틀·성장 → 보스전 → 탈출까지 한 판으로 완결
- **전투 깊이**: 직업별 스킬·다단히트·독 상태이상·치명타·3턴 보스 패턴
- **친숙한 재미**: "포켓몬식 충돌 배틀"이라는 누구나 아는 그립감
- **확장 여지**: 여러 층 던전, 장비·상점·골드, NPC·퀘스트가 자연스럽게 추가 가능
