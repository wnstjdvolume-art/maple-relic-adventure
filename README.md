# 🍁 Maple Relic Adventure

탑다운 던전 탐험 + **포켓몬 스타일 턴제 배틀** 게임. 메이플스토리 유니버스(MSU)
**바이브 캠프(MapleStory Vibe Camp)** 출품을 목표로 한 프로젝트.

> 필드를 돌아다니며 고대 유물 5개를 모으고, 몬스터와 **부딪히면 화면이 전환되며 턴제 배틀**이
> 시작된다. 유물을 모두 모으면 탈출 포탈이 열린다.

---

## 📁 구성

| 파일 | 설명 |
|---|---|
| `index.html` | **로컬 플레이 가능 프로토타입** (단일 파일, 더블클릭하면 브라우저에서 바로 플레이) |
| `MSU_VIBE_PROMPTS.md` | **MSU Space(Verse8)에 그대로 입력할 바이브코딩 프롬프트 묶음 + 게임 기획서** ← 실제 출품용 |
| `_test.js` | 게임 로직 헤드리스 검증 스크립트 (`node _test.js`) |

## ▶ 로컬 플레이 방법

`index.html`를 더블클릭 → 브라우저에서 실행. 설치 불필요.

- 시작 → **직업 선택**(⚔️전사 / 🔮마법사 / 🏹궁수)
- **필드 이동**: `WASD` 또는 화살표
- **배틀**: `↑ ↓`로 명령 선택, `Enter`/`Space` 확인, `Esc` 뒤로 (숫자 `1~4`도 가능)
- 명령: ⚔ 공격 / ✦ 스킬(직업별, 서브메뉴) / 🧪 물약 / 🏃 도망

## 🎯 목표

1. 직업을 골라 던전을 탐험하며 **유물 5개** 회수
2. 몬스터 **12종**과 부딪히면 턴제 배틀 → 처치하면 경험치·레벨업 (직업 스킬·독·치명타 활용)
3. 유물 5개를 모으면 수호자 보스 **머쉬맘** 등장 → 처치하면 포탈 개방
4. 포탈로 **B2 망각의 지하**로 진입(스탯 유지, HP·MP 회복+물약 보상) → 더 강한 몬스터 4종 추가
5. B2 유물 5개 → **최종 보스 자쿰**(4턴 패턴) 처치 → 탈출하면 **최종 클리어** 🏆

## ⚠️ MSU 바이브 캠프 출품에 대하여

이 게임의 **최종 출품물은 로컬 HTML 파일이 아니라 MSU Space(Verse8) 위에 만든 게임**이어야 한다.

- 대회 기간: **2026-06-08 ~ 06-29** (빌드), 심사 ~07-17, 수상발표 07-22
- 제작 방식: MSU Space의 **"바이브 코딩"** — 자연어로 설명하면 AI(Agent 8)가 게임을 생성
- **메이플스토리 공식 에셋만** 사용 가능 (캐릭터/몬스터/아이템/배경/BGM)
- 참가: vibecamp.msu.io → MSU Space 가입 → 크리에이터 프로그램 지원 → 제출 폼

이 로컬 프로토타입은 **"무엇을 만들지"를 확정하고 조작감을 검증하기 위한 기준 구현**이며,
`MSU_VIBE_PROMPTS.md`의 프롬프트를 MSU Space에 순서대로 입력해 동일한 게임을 재현한다.
